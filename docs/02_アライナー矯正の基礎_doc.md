# ドキュメント：02_アライナー矯正の基礎.md

## 対応ファイル
`02_claude/02_アライナー矯正の基礎.md`

## 概要
アライナー矯正（クリアアライナー）の基礎知識をアメリカ発のオープンアクセス論文に基づいてまとめた学習資料。

## インプット（情報源）
- PMC10597356: "Clear Aligner Therapy: Up to date review" (2023)
- PMC7018497: "Orthodontic Treatment with Clear Aligners and The Scientific Reality Behind Their Marketing" (2019)
- PMC8610387: "Efficacy of clear aligners in controlling orthodontic tooth movement" (2015)

## アウトプット
- Markdownファイル（画像7枚を含む学習資料）
- 対象: 歯科矯正初学者（専門用語+解説の形式）

## セクション説明

| セクション | 内容 |
|---|---|
| 1. アライナーの作り方 | 手作業セットアップ vs CAD-CAM |
| 2. バイオメカニクス | 変位駆動型 vs 力駆動型、アタッチメント |
| 3. 歯の動きの種類 | 傾斜移動、歯体移動、回転、挺出、圧下、トルク |
| 4. 有効性（エビデンス） | 各動きの正確性データ、Kravitz研究 |
| 5. 適応症と禁忌症 | 使えるケース/使えないケース |
| 6. 注意点 | 装着時間、歯根吸収、歯周組織、安定性 |
| 7. 治療前後比較 | 症例写真 |
| 8. まとめ | 重要な数字と用語一覧 |
