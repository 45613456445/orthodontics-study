# ドキュメント：01_矯正診断の流れ.md

## 対応ファイル
`02_claude/01_矯正診断の流れ.md`

## 概要
歯科矯正の診断プロセス全体を初学者向けに解説した学習資料。

## 内容構成

### インプット（情報源）
- NCBI StatPearls: Cephalometric Analysis (NBK594272)
- NCBI StatPearls: Malocclusion (NBK592395)
- PMC9687964: Learning Cephalometric Landmarks
- PMC9225851: Cephalometric Analysis Using AI
- 各種歯科矯正クリニックの解説ページ

### アウトプット
- Markdownファイル（画像7枚を含む学習資料）
- 対象: 歯科矯正初学者（11歳レベルの説明）

### セクション説明

| セクション | 内容 |
|---|---|
| ステップ1: 資料採得 | 診断に必要な5つの資料の説明 |
| ステップ2: セファロ分析 | 計測点8+5個、基準平面3つ、重要角度（SNA/SNB/ANB等） |
| ステップ3: 不正咬合の分類 | Angle Class I/II/III の解説（画像付き） |
| ステップ4: 模型分析 | アーチレングスディスクレパンシー等の分析項目 |
| ステップ5: 診断と治療計画 | 最終診断と治療方針の決定 |
| 用語一覧 | 基本用語21語の読み方と意味 |
