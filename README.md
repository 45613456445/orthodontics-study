# 矯正について。

歯科矯正学の初学者向け学習資料リポジトリ

## フォルダ構成

```
矯正について。/
├── 02_claude/           ← メインの学習資料
│   ├── 01_矯正診断の流れ.md   ← 矯正診断の全体フロー（用語・画像付き）
```

## 内容

### 01_矯正診断の流れ
- 資料採得（写真・レントゲン・模型）
- セファロ分析（計測点・基準平面・角度）
- 不正咬合の分類（Angle Class I/II/III）
- 模型分析（アーチレングスディスクレパンシー等）
- 診断と治療計画
