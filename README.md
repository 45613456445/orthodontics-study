## 内容

### 01_矯正診断の流れ
- 資料採得（写真・レントゲン・模型）
- セファロ分析（計測点・基準平面・角度）
- 不正咬合の分類（Angle Class I/II/III）
- 模型分析（アーチレングスディスクレパンシー等）
- 診断と治療計画

### 02_アライナー矯正の基礎
- アライナーの製造方法（手作業 vs CAD-CAM）
- バイオメカニクス（変位駆動型 vs 力駆動型）
- アタッチメントとパワーリッジ
- 各歯の動きの有効性データ（Rossini 2015, Kravitz研究）
- 適応症と禁忌症
- 主要3論文に基づくエビデンス

### 03_学習問題集
- Part 1：矯正診断の基礎（5問）
- Part 2：アライナー矯正の知識（6問）
- Part 3：Proffit先生の矯正学の基礎概念（5問）
- 回答は折りたたみ表示（detailsタグ）

### 04_Proffit_Contemporary_Orthodontics
- 第6版の全20章・7セクションの目次
- 第7版の変更点（アライナー章が新設）
- Ackerman-Proffit分類、問題志向型アプローチ
- 歯の動きの生物学的基礎、力学の原理
- オンライン学習リソース一覧
