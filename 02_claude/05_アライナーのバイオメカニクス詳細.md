# アライナーのバイオメカニクス詳細

> この資料は、PMC11570571「Seeking orderness out of the orderless movements」（Progress in Orthodontics, 2024）の内容をベースに、アライナー矯正のバイオメカニクス（力学）を詳しく解説したものです。

---

## この資料で学べること

1. アライナーの **素材と力学的特性**（どんな材料で、どう力が変わるか）
2. **力の伝え方のしくみ**（厚さ・トリミング・アタッチメント・TADs）
3. **各歯の動きのバイオメカニクス**（圧下・引き込み・トルク・回転・臼歯移動）
4. **研究方法**（FEA・物理センサー）

## 1. アライナーの素材と力学的特性

### 1-1. 基本原理

アライナーによる歯の移動は、2つの要素で決まります：

| 要素 | 英語 | かんたんに言うと |
|---|---|---|
| **適用される力** | Applied Force | アライナーが歯を押す力 |
| **歯周組織の生物学的反応** | Biological Response of Periodontal Tissues | 歯の周りの組織がどう反応するか |

**歯根膜（PDL）にかかる最適な圧力：4.7〜16 KPa**（キロパスカル）

> かんたんに言うと：強すぎず弱すぎない、ちょうどいい圧力の範囲がある

### 1-2. アライナーの理想的な条件

- **一定の力を持続的に発揮**（force delivery＝力の供給が安定していること）
- **適度な剛性**（rigidity＝硬さ）と高い降伏強度（yield strength＝壊れにくさ）
- **弾性範囲内**（elastic range＝元に戻れる範囲）で力を伝達する
- **Essix材料の最適な活性化範囲：0.2〜0.5 mm**

### 1-3. 素材の進化

| 年代 | 素材 | 特徴 |
|---|---|---|
| 1971年 | ポリメチルメタクリレート（PMMA） | 歯の移動が限定的 |
| 1985年 | 改良型インビジブルリテーナー | McNamaraが開発 |
| 1998年 | 単層ポリウレタン | 主要素材として使用 |
| 現在 | **PETG**、TPU、PET、PPC | 様々なポリマー |
| 最新 | **SmartTrack**、3Dプリント素材 | 新世代の素材 |

> **PETG**（Polyethylene Terephthalate Glycol＝ポリエチレンテレフタレートグリコール）が現在の主流素材

### 1-4. 主要素材の力学的特性の比較

#### 熱可塑性素材（Thermoplastic Materials）

| アライナー | 素材 | 弾性係数（MPa） | 応力緩和率 |
|---|---|---|---|
| Invisalign | SmartTrack (LD30) | 842 ± 63（乾燥）/ 61.3 ± 9.18（湿潤） | 低い（安定的） |
| Invisalign | EX30 | 103.2 ± 17.3 | 0.00941 |
| Biolon | PETG | 1,693 ± 51 | -- |
| Duran | PET | 2,366 | 0.0833 |
| F22 | ポリウレタン | 2,770 | 0.1183 |

> **弾性係数**（Elastic Modulus＝ヤング率）：数字が大きいほど硬い素材
> **応力緩和率**（Stress Relaxation Rate）：数字が大きいほど力が早く弱くなる

#### 3Dプリント素材

| 素材 | 弾性係数（MPa） | 極限引張強度（MPa） |
|---|---|---|
| Dental LT Clear resin | 1,540 ± 107 | -- |
| TC-85 | 1,186 | 20.7 ± 1.2 |
| OD-Clear TF | 38.4 ± 14.7 | 9.34 ± 1.96 |

> **3Dプリント素材のメリット**：熱成形時の幾何学的不正確さ（geometric inaccuracies）と収縮・膨張によるミスフィットを解消できる

### 1-5. 重要な力学的概念

#### 応力緩和（Stress Relaxation）

アライナーは **粘弾性ポリマー素材**（viscoelastic polymer）でできています。NiTi（ニッケルチタン）ワイヤーのような超弾性（superelastic）素材ではないため、時間が経つと力が弱くなります。

```
装着直後 → 強い力
  ↓ 時間経過
数時間後 → 力が徐々に弱くなる（指数関数的に減衰）
  ↓
1〜2週間後 → かなり力が弱くなっている
```

> これが「予測と実際の治療結果の差が約50%以上」になる大きな原因の一つ

#### SmartTrackの特徴

SmartTrack（Invisalignの素材）は **応力緩和が少なく**、より予測可能な歯の移動を実現：
- 良好な機械的特性（mechanical properties）
- 応力緩和を防ぐ特性
- 湿潤環境（口の中）でも安定

#### 多層素材（Multilayer Materials）の利点

PETG/TPUやTPU/PCの多層素材は：
- より安定した応力減衰速度（decay rate）
- 一貫性のある力の供給
- 単層素材より低い応力緩和率

## 2. 力の伝え方のしくみ（Force Transmission）

### 2-1. アライナーが歯を動かす基本メカニズム

アライナーは **ポリマーシェルの弾性回復力**（rebound deformation）で歯を動かします。

```
1. アライナーを歯にはめる
   ↓
2. アライナーの形と歯の形にズレがある（プログラムされた移動量）
   ↓
3. アライナーが元の形に戻ろうとする力で歯を押す
   ↓
4. 合力（resultant force）と抵抗中心（CR）の関係で歯の動き方が決まる
```

**アライナーが得意な動き：**
- **圧下**（intrusion＝歯を骨の中に押し込む）← アタッチメントなしでも可能
- **傾斜移動**（tipping＝歯冠が倒れる動き）← アタッチメントなしでも可能

**アライナーが苦手な動き：**
- **歯体移動**（translation＝歯全体を平行に動かす）
- **回転**（rotation＝歯を長軸周りに回す）
- **トルク**（torque＝歯根を動かす）

### 2-2. アライナーの厚さの影響

| 厚さ | 効果 |
|---|---|
| **薄いアライナー** | 傾斜移動（tipping）に適している |
| **厚いアライナー** | 歯体移動（translation）・歯根移動に必要 |
| **厚すぎるアライナー** | 副作用（side effects）が増加 |

> 力の発生量は **厚さと素材の組み合わせ** で決まる

### 2-3. トリミングライン（Trimming Line）のデザイン

| デザイン | 特徴 |
|---|---|
| **直線型**（Straight） | スカラップ型より大きな圧力を発生 |
| **スカラップ型**（Scalloped） | 圧力が小さい |
| **延長トリミング** | 歯への圧力が増大 |

### 2-4. アタッチメント（Attachments）

#### 2種類のアタッチメント

| 種類 | 英語 | 特徴 |
|---|---|---|
| **従来型** | Conventional | 長方形・斜面型・楕円形・半球形。サイズが決まっている |
| **最適化型** | Optimized | 歯の解剖学と目標の動きに合わせてカスタム設計。ClinCheckで自動配置 |

#### アタッチメントの重要な発見

- 水平方向の長方形アタッチメント → 近遠心方向のコントロールに優れる
- **舌側アタッチメント**（palatal surface＝上顎の裏側に貼る）→ 唇側より良い歯の動き
- **オーバーハンギングアタッチメント**（overhanging＝歯冠から突き出た形）→ PDLへのストレスが少ない

> **重要な注意点：** アタッチメントがあってもなくても、**前歯の舌側傾斜（lingual tipping）を完全に防ぐことは不可能**

### 2-5. TADs（Temporary Anchorage Devices＝歯科矯正用アンカースクリュー）

TADs（ミニスクリュー）とアライナーの組み合わせ：

| 項目 | データ |
|---|---|
| **臼歯遠心移動の最適な力** | 約4.5オンス（≒130 g） |
| **効果** | PDLストレスを軽減、より均一な力の分布 |
| **注意点** | 力が大きすぎると臼歯の傾斜・回転・ストレスが増大 |

#### 直接固定源 vs 間接固定源

| 方式 | 英語 | メリット | デメリット |
|---|---|---|---|
| **直接** | Direct anchorage | 中間部の圧下効果、前歯の挺出を軽減 | -- |
| **間接** | Indirect anchorage | 臼歯の近心移動をより良く防止 | 小臼歯の圧下で後方傾斜が悪化する可能性 |

## 3. 各歯の動きのバイオメカニクス

### 3-1. 圧下（Intrusion＝歯を骨の中に押し込む動き）

**重要な事実：**
- アライナーによる圧下は純粋な垂直方向ではなく、**後方・下方への変位＋傾斜を伴う**
- **犬歯が最大の垂直方向の力を受ける**（個別でも一括でも）
- 前歯個別の圧下 → 小臼歯に最大の挺出力が発生
- TADsを使った全弓圧下 → 後方部の圧下が前方部より大きい

**臼歯の圧下で起きること：**

```
アライナーで臼歯を圧下しようとすると...
  → アタッチメントの位置に関係なく、反時計回りのモーメントが発生
  → 第二大臼歯の近心傾斜が起きる（コントロール不能）
```

### 3-2. 前歯の一括引き込み（En-masse Retraction）

抜歯後のスペースを閉じるために前歯をまとめて後ろに引くとき：

| 歯 | 受ける力の方向 | 結果の動き |
|---|---|---|
| 中切歯 | 最大の舌側方向の力 | 舌側傾斜＋挺出 |
| 犬歯 | 最大の遠心方向の力 | 遠心傾斜＋挺出 |
| 第二小臼歯 | 近心方向の力 | 近心傾斜＋圧下 |
| 第一大臼歯 | 近心方向の力 | 近心傾斜＋舌側傾斜 |

#### オーバートリートメント（Overtreatment＝過剰補正）の効果

```
オーバートリートメントなし：
  → ストレスが頬側の歯頸部・根尖部に集中
  → 舌側の非根尖部にも集中

オーバートリートメントあり（前方装置を反時計回りに回転）：
  → ストレスが歯列全体に均一に分布
  → 歯体移動に近づく
  → 歯根吸収のリスクが軽減
```

> **臨床的意義：** オーバートリートメントは不正確さを補正するだけでなく、力の分布を改善する重要な手法

### 3-3. 圧下＋一括引き込みの併用

スペース閉鎖時の副作用（トルクロス・前歯の不要な挺出）を対処するため：
- 切歯・犬歯への唇舌方向の力が増大
- 犬歯への遠心方向の力が減少
- 小臼歯への近心方向の力が増大
- **PDLストレスがより均一に分布**
- **前歯の引き込みが減り、圧下が増える** → バランスの取れた動き

### 3-4. トルク（Torque＝歯根の唇舌方向の移動）

> **「一般的に言って、アライナーはトルク移動が苦手です」**（論文の原文より）

トルクには **カップル**（couple＝大きさが等しく方向が反対の2つの平行な力）が必要ですが、アライナー単体ではカップルを発生させることが非常に困難です。

#### パワーリッジ（Power Ridge）のメカニズム

```
ステップ1: パワーリッジの接触点で歯冠に力F1がかかる → 舌側傾斜
ステップ2: 傾斜した歯冠がアライナー内面を押す → 追加の力F2が発生

(A) F1 = F2 の場合 → 純粋なカップル（理想的なトルク）
(B) F1 > F2 の場合 → カップル＋追加の力
(C) F1 < F2 の場合 → カップル＋追加の力
```

**パワーリッジの効果：**
- 水平楕円型アタッチメントよりも大きなトルク値を発揮
- 垂直方向の力成分を軽減
- パワーリッジの高さが増すと → 歯体移動に近づく

#### パワーアーム（Power Arm）の効果

パワーアームの長さが増すと → 合力が抵抗中心（CR）に近づく → トルク移動が減少

### 3-5. 回転（Rotation＝歯を長軸周りに回す動き）

| 方向 | 必要なモーメント |
|---|---|
| **近心回転**（mesial rotation） | 小さいモーメントで可能 → 臨床的に容易 |
| **遠心回転**（distal rotation） | 大きいモーメントが必要 |

**アタッチメントなしの場合：**
- トルクが大幅に減少
- 回転に伴って **最も頻繁に圧下（intrusion）が起きる**

**アタッチメントありの場合：**
- モーメントが増加
- 圧下の発生が減少

### 3-6. 臼歯の遠心移動（Molar Distalization＝臼歯を後ろに動かす）

- 臼歯根にかかる引張・圧縮力は前歯より **著しく大きい**
- 中切歯にかかるストレスは最低
- **後方歯のPDLストレスは前方歯より著しく大きい**
- TADsとエラスティックを使うと → PDLストレスが軽減、より均一な分布

### 3-7. 臼歯の近心移動（Molar Mesialization＝臼歯を前に動かす）

> **「アライナーで最も困難な歯の動きの一つ」**、特に長距離の場合

| 段階 | 起きること |
|---|---|
| **初期** | 近心歯根の歯頸部＋遠心歯根の根尖部に圧縮応力 |
| **中間期** | 舌側回転＋臼歯の圧下（予想外の動き） |
| **終了時** | 近心に圧縮、遠心に引張 |

## 4. 研究方法

### 4-1. 有限要素解析（FEA：Finite Element Analysis）

最も一般的に使用される研究手法。コンピュータシミュレーションで力の分布を可視化できます。

#### FEAで使用される材料パラメータ

| 組織 | 弾性係数（MPa） | ポアソン比 |
|---|---|---|
| **エナメル質** | 84,100 | 0.33 |
| **象牙質** | 14,700 | 0.31 |
| **歯（未分化）** | 19,600 | 0.30 |
| **海綿骨** | 1,370 | 0.30 |
| **皮質骨** | 13,700〜34,500 | 0.30 |
| **歯根膜（PDL）** | 0.67〜0.69（一般的）; 50（Cattaneo） | 0.45 |
| **アライナー** | 528（一般的）; 816 | 0.36 |

> エナメル質の弾性係数は84,100 MPa、アライナーは528 MPa → 歯はアライナーの約160倍硬い

#### FEAの限界

- 抵抗中心（CR）の位置を正確に設定するのが実質的に不可能
- 接触パラメータ（面積・圧力分布・接触点）が非常に変動的
- シミュレーションでは均一な厚さを想定しているが、実際は歯の表面で厚さが異なる
- パラメータ設定の精度が不確実 → 結果の信頼性に疑問

### 4-2. 物理的計測システム

| 年代 | システム | 特徴 |
|---|---|---|
| 1992年 | 3次元トルク測定装置 | Planert et al. |
| 2009年 | OSIM シミュレーター | Nano17センサー使用、精度0.01 mm |
| 2018年 | FORSED | 14本の下顎歯の3軸力・トルクを同時測定 |
| 2021年 | Kaur et al. | 0.75 mmアライナーの上顎歯への力を測定 |
| 2023年 | Hu et al. | Nano17で臼歯遠心移動の3Dメカニカルモデル |

> **Nano17センサー**：コンパクトで精度が非常に高い6軸力・トルクセンサー。アライナー研究で最も選ばれるセンサー

## 5. まとめ：アライナーのバイオメカニクス 13のポイント

| # | ポイント |
|---|---|
| 1 | アライナーは**ポリマーシェルの弾性回復力**で歯を動かす（ワイヤーとは根本的に異なる） |
| 2 | アライナーは本質的に**圧下と傾斜移動が得意**（アタッチメントなしでも可能） |
| 3 | **歯体移動・回転・トルクは苦手**（アタッチメントやパワーリッジが必要） |
| 4 | 予測と実際の治療結果の差は**約50%以上**（応力緩和とクリープが主因） |
| 5 | **応力緩和が核心的な問題**：NiTiワイヤーと違い、力が時間とともに減衰する |
| 6 | アタッチメントは必要だが不完全：**前歯の舌側傾斜は完全には防げない** |
| 7 | **パワーリッジがトルクの最良の手段**だが、それでも限界がある |
| 8 | **オーバートリートメントが重要**：力の分布を改善し、歯体移動を促進する |
| 9 | **TADsが治療効果を大幅に向上**（臼歯遠心移動の最適力≒130g） |
| 10 | **臼歯の近心移動が最も困難な動き** |
| 11 | **歯根の移動は歯冠の移動よりはるかに困難**（根本的な課題） |
| 12 | 3Dプリント素材（DLP技術）が熱成形の不正確さを解消する可能性があるが、まだ商用化されていない |
| 13 | 患者のコンプライアンス（**1日20〜22時間の装着**）が不可欠 |

## 用語集（Glossary）

### 力に関する用語

| 日本語 | 英語 | 意味 |
|---|---|---|
| 矯正力 | Orthodontic Force | 装置が歯を動かすために加える力 |
| 舌側力 | Lingual Force | 舌の方向への力 |
| 唇側力 | Labial Force | 唇の方向への力 |
| 遠心力 | Distal Force | 正中線から離れる方向の力 |
| 近心力 | Mesial Force | 正中線に向かう方向の力 |
| 挺出力 | Extrusive Force | 歯を歯槽から引き出す力 |
| 圧下力 | Intrusive Force | 歯を歯槽に押し込む力 |
| カップル | Couple | 大きさが等しく方向が反対の2力（純粋な回転を生む） |
| 合力 | Resultant Force | すべての力を合成した正味の力 |

### 歯の動きの種類

| 日本語 | 英語 | 意味 |
|---|---|---|
| 傾斜移動 | Tipping | 歯冠が動き、歯根はほぼ動かない（アライナーが最も得意） |
| 歯体移動 | Translation / Bodily Movement | 歯全体が同じ方向に動く（アライナーは苦手） |
| 回転 | Rotation | 歯が長軸の周りに回る |
| 圧下 | Intrusion | 歯を骨の中に押し込む（アライナーが得意） |
| 挺出 | Extrusion | 歯を骨の外に引き出す |
| トルク | Torque | 歯根の唇舌方向の移動（アライナーが最も苦手） |
| 臼歯遠心移動 | Molar Distalization | 臼歯を後ろに動かす |
| 臼歯近心移動 | Molar Mesialization | 臼歯を前に動かす（最も困難） |
| 一括引き込み | En-masse Retraction | 複数の前歯を同時に後ろに引く |

### 素材・力学的特性

| 日本語 | 英語 | 意味 |
|---|---|---|
| 弾性係数 | Elastic Modulus（Young's Modulus） | 素材の硬さの指標 |
| 極限引張強度 | Ultimate Tensile Strength（UTS） | 素材が壊れる前の最大応力 |
| 降伏強度 | Yield Strength | 永久変形が始まる応力 |
| 応力緩和 | Stress Relaxation | 一定の変形のもとで力が時間とともに弱くなる現象 |
| クリープ | Creep | 一定の力のもとで変形が時間とともに大きくなる現象 |
| 粘弾性 | Viscoelastic Behavior | 弾性と粘性を併せ持つ時間依存的な性質 |
| ヒステリシス | Hysteresis Effect | 負荷と除荷の応力-歪み曲線の差 |
| ガラス転移温度 | Glass Transition Temperature | ポリマーが硬い状態から柔軟な状態に変わる温度 |

### 解剖学・構造に関する用語

| 日本語 | 英語 | 意味 |
|---|---|---|
| 抵抗中心 | Center of Resistance（CR） | 力をかけると回転なしで純粋な移動が起きる点 |
| 歯根膜 | Periodontal Ligament（PDL） | 歯根と歯槽骨をつなぐ結合組織 |
| 歯槽骨 | Alveolar Bone | 歯根を取り囲む骨 |
| 皮質骨 | Cortical Bone | 骨の外側の硬い層 |
