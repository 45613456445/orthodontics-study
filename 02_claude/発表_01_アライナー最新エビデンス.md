## 論文1：国際デルファイ合意 ── アライナーの適応症と限界

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Indications and limits of clear aligner therapy: an international modified Delphi consensus study |
| **著者** | D'Anto V, Oliva G, Nieri M, Franchi L, + 国際専門家23名 |
| **米国パネリスト** | James Mah（UNLV）、Jean-Marc Retrouvey（Baylor）、Flavio Uribe（UConn） |
| **ジャーナル** | Progress in Orthodontics, 2025 |
| **PMID** | 40760282 / **PMC12321729**（全文無料） |
| **研究デザイン** | 修正デルファイ法（25項目のステートメントに対する合意形成） |

### 主要な結果

**合意が得られた項目（25項目中22項目で合意）：**

#### アライナーが有効なケース
| ステートメント | 合意率 |
|---|---|
| Class I 非抜歯ケース（軽度〜中等度の叢生） | **91%** |
| 特定の開咬（open bite）ケース | 合意あり |
| 歯周組織の健康維持（oral hygiene＝口腔衛生）の面で優位 | 合意あり |
| QOL（Quality of Life＝生活の質）の面で優位 | 合意あり |

#### アライナーの限界
| ステートメント | 合意率 |
|---|---|
| アライナーは主に**歯冠の傾斜移動（crown tipping）**を起こす（歯体移動ではない） | **96%** |
| 骨格的な拡大では**急速拡大装置（RPE）がアライナーに勝る** | **96%** |
| 重度の抜歯ケースでは固定装置に比べて結果が劣る | 合意あり |

#### 合意が得られなかった項目（3項目）
- 歯根吸収のリスク
- 治療後の後戻り（relapse）の程度
- 治療期間の比較

### 発表でのポイント
> **「アライナーは軽度〜中等度のケースでは有効。しかし、歯体移動ではなく歯冠傾斜が主な動きであることを96%の専門家が認めている」**

## 論文2：アライナー vs 固定装置 ── 21のRCTに基づくメタアナリシス

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Effectiveness of Clear Orthodontic Aligners in Correcting Malocclusions: A Systematic Review and Meta-Analysis |
| **著者** | Baneshi M, O'Malley L, El-Angbawi A, Thiruvenkatachari B |
| **所属** | マンチェスター大学（英国） |
| **ジャーナル** | Journal of Evidence-Based Dental Practice, 2025 |
| **PMID** | 39947778 |
| **研究デザイン** | 系統的レビュー＋メタアナリシス（**21のRCT、970名**） |

### 主要な結果

#### アライナーと固定装置で差がなかった項目
| 評価指標 | 結果 |
|---|---|
| ABO OGS スコア（治療結果の客観評価） | **有意差なし** |
| Little's Irregularity Index（歯列不正の指標） | **有意差なし** |
| PAR スコア（咬合の評価） | **有意差なし** |

> 単純な不正咬合に対しては、アライナーと固定装置の治療効果は**同等**

#### アライナーが優れていた項目
| 項目 | データ |
|---|---|
| **プラークインデックス** | アライナー群で有意に低い |
| **歯肉炎インデックス** | アライナー群で有意に低い |
| **BOP（出血指数）** | アライナー群で有意に低い |
| **QOL（6ヶ月時点）** | アライナー群で有意に高い |
| **チェアタイム** | アライナー群で有意に短い |
| **歯根吸収** | アライナー群で有意に少ない |

### 発表でのポイント
> **「21のRCT（970名）のメタアナリシスで、単純な不正咬合ではアライナーと固定装置の治療効果は同等。さらに、歯周組織の健康・QOL・歯根吸収の面ではアライナーが優位」**

## 論文3：抜歯ケースでのアライナー ── 系統的レビュー＋メタアナリシス

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Clear Aligners in Extraction-Based Orthodontic Treatment: A Systematic Review and Meta-Analysis |
| **著者** | Abu Arqub S, Chheda H, Crumpton J, Yang X, Farha P, Upadhyay M |
| **所属** | フロリダ大学 / コロンビア大学 / コネチカット大学（米国） |
| **ジャーナル** | Orthodontics and Craniofacial Research, **2026年2月** |
| **PMID** | 41195763 |
| **研究デザイン** | 系統的レビュー＋メタアナリシス（**20研究、536名アライナー vs 173名固定装置**） |

### 主要な結果

| 項目 | 結果 |
|---|---|
| **予測と実際の差** | 抜歯ケースで有意な差がある |
| **臼歯の過度な傾斜** | アライナー群で顕著 |
| **前歯の後方移動不足** | 計画通りに達成されない |
| **歯根コントロール** | 固定装置に比べて**劣る** |
| **咬合接触スコア** | 固定装置に比べて**劣る** |

#### 対策として推奨
- 戦略的なオーバーコレクション（overcorrection＝過剰補正）
- 補助的なメカニクス（TADs、エラスティック等）の併用
- リファインメント（refinement＝追加ステージ）の計画

### 発表でのポイント
> **「抜歯ケースでは、アライナーは予測通りの動きを達成できないことが多い。特に歯根コントロールと臼歯の傾斜制御が課題。オーバーコレクションと補助メカニクスが必須」**

## 論文4：Class II 臼歯遠心移動 ── 系統的レビュー

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Effectiveness and accuracy of clear aligners in non-extraction Class II correction: a systematic review of maxillary molar distalization and rotation in the permanent dentition |
| **著者** | Disthaporn S, Allareddy V, Atsawasuwan P, Lee MK |
| **所属** | イリノイ大学シカゴ校 矯正歯科（米国） |
| **ジャーナル** | Angle Orthodontist, 2025 |
| **PMID** | 41015416 / **PMC12544393**（全文無料） |
| **研究デザイン** | 系統的レビュー（**16研究、2015〜2024年**） |

### 主要な結果

| 項目 | データ |
|---|---|
| **遠心移動量** | 1.84〜2.98 mm |
| **遠心移動の最大正確性** | **85%** |
| **第一大臼歯の回転** | 約8度 |
| **回転の正確性** | **78.4%** |

#### 成功のための条件
- **Sequential distalization（順次遠心移動）**プロトコル
- リファインメントの計画
- 適切なアタッチメントの使用

#### 課題
- **固定源の喪失**（anchorage loss＝前歯が前に出てしまう副作用）
- コンプライアンス（装着時間）のモニタリング

### 発表でのポイント
> **「アライナーでのClass II遠心移動は最大85%の正確性。2〜3mmの遠心移動が達成可能。ただし固定源の喪失とコンプライアンスに注意が必要」**

## 論文5：開咬（Open Bite）の治療 ── 系統的レビュー＋メタアナリシス

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Efficacy of Clear Aligner Therapy for the Treatment of Anterior Open Bite in Adults: A Systematic Review and Meta-Analysis |
| **著者** | Correa E, Michelogiannakis D, Barmak AB, Rossouw PE, Javed F |
| **所属** | ロチェスター大学 Eastman Institute for Oral Health（米国） |
| **ジャーナル** | Orthodontics and Craniofacial Research, 2025 |
| **PMID** | 39445659 |
| **研究デザイン** | 系統的レビュー＋メタアナリシス（**14研究、12研究をメタアナリシスに含む**） |

### 主要な結果

| 項目 | データ |
|---|---|
| **開咬の改善量** | 平均 **2.76 mm**（95%CI: 2.23〜3.28） |
| **改善のメカニズム** | 主に**切歯の挺出**（extrusion）による |
| **臼歯の圧下** | 有意な変化なし |
| **下顎平面角の変化** | 有意な変化なし |
| **エビデンスの信頼度** | 高い |

### 発表でのポイント
> **「成人の前方開咬に対して、アライナーは平均2.76mmの改善を達成。ただし、改善は主に切歯の挺出によるもので、臼歯の圧下ではないことに注意」**

## 論文6：過蓋咬合（Deep Bite）の治療 ── 系統的レビュー

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Effectiveness and accuracy of clear aligners in treatment of deep bite: a systematic review |
| **著者** | Husain F, Riccardi J, Alletto L, Stellrecht E, Bejehmir AP, Al-Jewair T |
| **所属** | バッファロー大学 矯正歯科（米国） |
| **ジャーナル** | Clinical Oral Investigations, 2025 |
| **PMID** | 40394411 |
| **研究デザイン** | 系統的レビュー（**18研究**） |

### 主要な結果

| 項目 | データ |
|---|---|
| **咬合の開口量** | 0.4〜3.8 mm |
| **正確性（accuracy）** | **33%〜48.88%** |
| **有効なケース** | 軽度〜中等度の歯槽性（dentoalveolar）過蓋咬合 |
| **限界** | 骨格性の過蓋咬合では固定装置に劣る |
| **エビデンスの質** | 低い |

### 発表でのポイント
> **「過蓋咬合の正確性は33〜49%と低い。軽度〜中等度の歯槽性ケースには有効だが、骨格性のケースでは固定装置を検討すべき」**

## 論文7：3Dプリントアライナー ── 12ヶ月の臨床評価

### 基本情報

| 項目 | 内容 |
|---|---|
| **タイトル** | Directly printed aligner therapy: A 12-month evaluation of application and effectiveness |
| **著者** | Knode V, Ludwig B, Retrouvey JM, Pandis N, Schmid JQ, Erbe C, Fleming PS |
| **所属** | マインツ大学（独）、Baylor（米）、トリニティカレッジ（アイルランド）、ベルン大学（スイス） |
| **ジャーナル** | American Journal of Orthodontics and Dentofacial Orthopedics (AJODO), 2025 |
| **PMID** | 39425729（無料記事） |
| **研究デザイン** | 後方視的評価（**54名、12ヶ月**） |

### 主要な結果

| 項目 | データ |
|---|---|
| **PARスコアの改善** | 17.01 → 2.25（**86.6%改善**） |
| **リファインメントが必要だった割合** | **40.8%** |
| **合併症** | わずか3名（軽度） |
| **対象** | 軽度〜中等度の不正咬合 |

#### 3Dプリントアライナーの特徴
- 従来の熱成形（thermoforming）ではなく、直接3Dプリントで製造
- 形状記憶（shape memory）特性
- より一貫した力のプロファイル（force profile）
- 精度・正確性が熱成形型より高い

### 発表でのポイント
> **「3Dプリントで直接製造されたアライナーが86.6%のPAR改善を達成。将来は熱成形型に代わる可能性があるが、まだ大規模RCTが必要」**

## エビデンスの全体像 ── 一目でわかるまとめ

### アライナーが得意なこと（エビデンスあり）

| ケース | 有効性 | 根拠 |
|---|---|---|
| 軽度〜中等度の叢生（非抜歯） | 固定装置と同等 | Baneshi: 21 RCT, 970名 |
| 歯周組織の健康維持 | 固定装置より優位 | Baneshi: プラーク・歯肉炎・BOPが低い |
| 患者のQOL | 固定装置より優位 | Baneshi: 6ヶ月時点で有意差 |
| Class II 臼歯遠心移動 | 最大85%の正確性 | Disthaporn: 16研究 |
| 前方開咬 | 平均2.76mmの改善 | Correa: 14研究 |
| 歯根吸収のリスク | 固定装置より少ない | Baneshi: メタアナリシス |

### アライナーが苦手なこと（エビデンスあり）

| ケース | 問題点 | 根拠 |
|---|---|---|
| 抜歯ケース | 歯根コントロール不良、臼歯傾斜 | Abu Arqub: 20研究, 536名 |
| 過蓋咬合 | 正確性33〜49%と低い | Husain: 18研究 |
| 歯体移動 | 96%の専門家が「主に歯冠傾斜」と認定 | D'Anto: Delphi合意 |
| 骨格性の拡大 | RPE（急速拡大装置）に劣る | D'Anto: Delphi合意 |
| 重度の抜歯ケース | 固定装置に比べて結果が劣る | D'Anto: Delphi合意 |

### まだわからないこと（エビデンス不足）

| 項目 | 状況 |
|---|---|
| 歯根吸収の長期リスク | 専門家間で合意が得られず（Delphi） |
| 後戻り（relapse）の程度 | 専門家間で合意が得られず（Delphi） |
| 治療期間の正確な比較 | データ不十分 |
| 骨格性の過蓋咬合への対応 | エビデンスの質が低い |

## Take-Home Messages（持ち帰りメッセージ）

### 1. アライナーは「道具の一つ」である
アライナーは万能ではなく、適切なケース選択（case selection）が成功の鍵。軽度〜中等度の非抜歯ケースでは固定装置と同等の効果がある。

### 2. 歯冠傾斜 ≠ 歯体移動
アライナーが起こす動きは主に**歯冠の傾斜（tipping）**であり、歯体移動（bodily movement）ではない。この事実を96%の国際専門家が認めている。

### 3. 抜歯ケースは要注意
予測通りの動きが達成されないリスクが高い。オーバーコレクション、TADs、エラスティックなどの補助メカニクスを最初から計画に組み込む。

### 4. 歯周組織の健康に有利
プラーク・歯肉炎・BOP・歯根吸収のすべてでアライナーが固定装置より有利。患者のQOLも高い。

### 5. 3Dプリントが未来を変える可能性
直接3Dプリントアライナーが臨床で使われ始めている。形状記憶・一貫した力・高精度という特徴を持つが、まだRCTが不足している。
