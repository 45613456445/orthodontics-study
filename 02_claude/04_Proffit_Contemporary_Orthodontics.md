## Proffit先生の重要な概念

### 1. Ackerman-Proffit分類（1969年）

Angleの分類（Class I/II/III＝前後方向だけ）を拡張して、**5つの要素で3次元的に評価**：

| # | 評価要素 | 英語 | 見ること |
|---|---|---|---|
| 1 | **歯の整列** | Alignment | 叢生・空隙・歯列の配列 |
| 2 | **横顔のプロファイル** | Profile | 凸型・直型・凹型 |
| 3 | **横方向のズレ** | Transverse | 交叉咬合・歯列弓の幅 |
| 4 | **垂直方向のズレ** | Vertical | 開咬・過蓋咬合 |
| 5 | **前後方向のズレ** | Anteroposterior | Angle Class I/II/III |

→ ベン図（Venn Diagram）を使って複数の問題が重なるケースを表現

### 2. 問題志向型アプローチ（Problem-Oriented Approach）

Proffit先生が推奨する診断の流れ：

```
1. 包括的なデータベースを構築（資料採得）
     ↓
2. 問題リスト（Problem List）を作成
     ↓
3. 治療戦略（Treatment Strategy）を決定
     ↓
4. 治療を実施
```

### 3. 歯の動きの生物学的基礎

| ステップ | 何が起きる？ |
|---|---|
| 1. 力の適用 | 歯に矯正力がかかる |
| 2. PDLの感知 | 歯根膜（PDL）が力を検知 |
| 3. 炎症反応 | 急性炎症反応が起きる |
| 4. サイトカイン放出 | RANKL等の信号物質が放出される |
| 5. 圧迫側：骨吸収 | 破骨細胞（osteoclast）が骨を溶かす |
| 6. 牽引側：骨添加 | 骨芽細胞（osteoblast）が新しい骨を作る |
| 7. 歯の移動 | 骨のリモデリングにより歯が動く |

**重要な原理：**
- **持続的で優しい力**（gentle, persistent force）が最も効果的
- 強すぎる力 → 歯根吸収やヒアリン化（PDLの壊死）のリスク
- 間欠的な力（intermittent force）でも歯は動く（PDLへのダメージが少ない）

### 4. 力学の基本原理

| 用語 | 英語 | 意味 |
|---|---|---|
| **力** | Force | 歯を押す・引く力 |
| **モーメント** | Moment | 歯を回転させる力 |
| **カップル** | Couple | 2つの平行で反対方向の力（純粋な回転を生む） |
| **抵抗中心** | Center of Resistance | 歯が回転する中心点 |
| **回転中心** | Center of Rotation | 実際に歯が回転する点 |
| **M/F比** | Moment-to-Force Ratio | 歯の動き方を決める力とモーメントの比率 |

## 学習の進め方（推奨）

### ステップ1：基礎概念（まずここから）
- Ch.1：不正咬合の分類と疫学
- Ch.2-4：成長発育の概念
- Ch.5：原因論

### ステップ2：診断（次にここ）
- Ch.6：セファロ分析・模型分析を含む診断法
- Ch.7：治療計画の立て方

### ステップ3：メカニクス（ここが難しいが重要）
- Ch.8：生物学的基礎
- Ch.9：力学の原理
- Ch.10：矯正装置の種類

### ステップ4：臨床治療（実践）
- Ch.11-14：小児の治療・成長コントロール
- Ch.15-18：包括的治療（整列→スペース閉鎖→仕上げ→保定）

### ステップ5：特殊な治療
- Ch.19-20：成人矯正・外科矯正
